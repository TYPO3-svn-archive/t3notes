Extension: t3notes

- lib/
Offers a libary for authentication to a Lotus Notes Server

- sv1/
Offers a auth service for the authentication of a fe_user or general website user to Lotus Notes Server through the libary above

- ext_autoload.php
Makes sure that the class in lib/class.tx_t3notes_auth.php is loaded once only


Use 'options.disableNotesTab' as UserTS option for disabling the notes tab
